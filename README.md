# Fear-and-hunger-Mod
I put my mod here
------
This mod is a small personal project made just for fun.
It allows you to open the Save Menu anywhere in the game by pressing a single key.

FEATURE:
- Press the "O" key to force open the Save Menu.
- Works on any map.

INSTALLATION:
1. Put the "Den_SaveMod.js" file into the folder: www/js/plugins/
2. Open the file "www/js/plugins.js" with a text editor.
3. Add this line to the plugin list:
   {"name":"Den_SaveMod","status":true,"description":"","parameters":{}}
